# 🐾 반려동물 실종 지도 - 사용자 계정 시스템

## 📋 프로젝트 개요

반려동물 강아지 실종 지도 프로젝트에 **사용자 계정 시스템**이 추가되었습니다.
모든 페이지는 하얀색(White) 디자인으로 통일되어 있습니다.

---

## 📁 파일 구조

### 생성된 파일 (새로 추가됨)

```
c:\spring-tool\
├── auth.php                 ✅ 회원 가입 / 로그인 페이지
├── profile.php              ✅ 개인 프로필 페이지
├── manage-reports.php       ✅ 내 신고/제보 관리 페이지
└── README.md                ✅ 이 파일
```

### 기존 파일 (수정됨)

```
c:\spring-tool\
└── index.php                ⚡ 사용자 메뉴 추가
```

---

## 🎨 디자인 특징

- **색상 스키마**: 흰색(White) 기본 배경 + 청록색(#4ecdc4) 강조
- **반응형**: 모든 페이지가 반응형 디자인으로 구현
- **사용자 친화적**: 직관적인 UI/UX
- **일관성**: 전체 시스템에서 일관된 디자인 언어 사용

---

## 📝 각 페이지 설명

### 1️⃣ `auth.php` - 회원 가입 / 로그인
**URL**: `http://localhost/spring-tool/auth.php`

**기능**:
- ✅ 회원 가입 폼 (이름, 이메일, 전화, 비밀번호, 주소)
- ✅ 로그인 폼 (이메일, 비밀번호)
- ✅ 비밀번호 보이기/숨기기 토글
- ✅ 로그인 상태 유지 옵션
- ✅ 비밀번호 확인 검증

**세션 저장**:
```javascript
sessionStorage.setItem('userEmail', email);
sessionStorage.setItem('userName', name);
```

---

### 2️⃣ `profile.php` - 개인 프로필
**URL**: `http://localhost/spring-tool/profile.php`

**탭 구성**:

#### 🔹 프로필 탭
- 사용자 정보 표시 (이름, 이메일, 전화, 주소, 가입 날짜)
- 프로필 사진 변경 (이미지 파일 업로드)
- 프로필 정보 수정 폼

#### 🔹 신고/제보 탭
- 내 신고/제보 목록 (격자형)
- 신고/제보 통계 (총 수, 진행 중, 해결됨)
- 상세보기, 수정, 삭제 기능
- 상태 필터링

#### 🔹 설정 탭
- 비밀번호 변경
- 알림 설정 (이메일, SMS)
- 계정 삭제 (경고 포함)

---

### 3️⃣ `manage-reports.php` - 내 신고/제보 관리
**URL**: `http://localhost/spring-tool/manage-reports.php`

**기능**:

#### 📊 통계 섹션
- 총 신고/제보 수
- 대기 중, 진행 중, 해결됨 현황

#### 🔍 필터 섹션
- 상태별 필터 (대기 중, 진행 중, 해결됨, 종료됨)
- 기간별 필터 (지난 1주, 1개월, 3개월)
- 텍스트 검색 (제목, 내용)

#### 👁️ 보기 모드
- **격자형 보기**: 카드 레이아웃 (기본값)
- **목록형 보기**: 테이블 레이아웃

#### ✏️ 신고/제보 관리
- 새 신고/제보 작성
- 신고/제보 상세보기
- 신고/제보 수정
- 신고/제보 삭제

#### 📝 신고/제보 양식
```
- 제목 (필수)
- 유형 (실종/목격/발견/부상/기타)
- 위치 (필수)
- 날짜 (필수)
- 상세 내용 (필수)
- 사진 (선택)
- 연락처 (선택)
```

---

### 4️⃣ `index.php` - 메인 페이지 (수정)
**추가된 기능**:
- 우상단에 사용자 메뉴 추가
- 사용자 로그인 상태 표시
- 프로필 이동 버튼
- 신고 관리 이동 버튼
- 로그아웃 버튼

---

## 🔐 사용자 인증 흐름

```
1. auth.php 접속
   ↓
2. 회원가입 또는 로그인
   ↓
3. SessionStorage에 사용자 정보 저장
   ├── userEmail: 사용자 이메일
   └── userName: 사용자 이름
   ↓
4. profile.php 또는 manage-reports.php로 자동 이동
   ↓
5. 로그아웃 시 SessionStorage 초기화
```

---

## 🚀 시작하기

### 1️⃣ 로그인 / 회원 가입
```
1. http://localhost/spring-tool/auth.php 접속
2. 회원가입 또는 로그인
3. 자동으로 프로필 페이지로 이동
```

### 2️⃣ 프로필 관리
```
1. 프로필 페이지에서 개인 정보 확인
2. "프로필 수정" 버튼으로 정보 변경
3. "사진 변경"으로 프로필 이미지 업로드
```

### 3️⃣ 신고/제보 관리
```
1. "내 신고/제보" 탭으로 이동
2. "+ 새 신고/제보" 버튼으로 새 신고 작성
3. 상태별/기간별로 필터링
4. 신고/제보 수정, 삭제
```

### 4️⃣ 메인 지도에서
```
1. 우상단 "프로필" 버튼 → 프로필 페이지
2. 우상단 "내 신고" 버튼 → 신고/제보 관리
3. 우상단 "로그인" 버튼 → 인증 페이지
4. 로그인 후 "로그아웃" 버튼 표시
```
