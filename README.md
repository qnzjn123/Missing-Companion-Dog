## 📋 프로젝트 개요## 📋 프로젝트 개요



**반려동물 실종 찾기**는 실종된 반려동물을 찾기 위한 통합 플랫폼입니다. 반려동물 강아지 실종 지도 프로젝트에 **사용자 계정 시스템**이 추가되었습니다.

사용자는 실종 신고를 하거나 목격한 반려동물을 제보할 수 있으며, 지도 기반 시스템으로 쉽게 검색할 수 있습니다.모든 페이지는 하얀색(White) 디자인으로 통일되어 있습니다.
